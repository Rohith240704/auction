This project was developed for an entrepreneur to digitize and manage an online auction system for buying and selling antique items.
Auction is a web-based online auction platform designed and developed for an entrepreneur operating in the antiques and collectibles domain. The platform helps transform traditional antique trading into a modern, secure, and scalable digital auction system.
Developed an online auction website for antiques for an entrepreneur, enabling real-time bidding, secure listings, and automated auction management.
